### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AbdallahRaslan/ExpensesTracker.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

3. Run Start 
  ```sh
   npm install
   ```
